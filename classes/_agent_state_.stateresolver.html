resolveLocalsList_

  • resolveLocalsList_(frame: FrameMirror, args: any): ScopeMirror[]
    • Defined in agent/state.ts:387

    Iterates and returns variable information for all scopes (excluding global) in a given frame. FrameMirrors should return their scope object list with most deeply nested scope first so variables initially encountered will take precedence over subsequent instance with the same name - this is tracked in the usedNames map. The argument list given to this function may be manipulated if variables with a deeper scope occur which have the same name.
